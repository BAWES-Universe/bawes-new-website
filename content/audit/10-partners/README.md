## WorkAdventure (Source Fork)

- **Relationship:** Universe is built on a fork of WorkAdventure by TheCodingMachine
- **Status:** Active partnership — credited in the Beyond the Avatar blog post and on bawes.net
- **Upstream:** `github.com/thecodingmachine/workadventure` (MIT open source)
- **Website:** workadventu.re
- **CEO:** Grégory (contact at gregory@workadventu.re)
- **Clients:** Allianz, Bosch, Capgemini, Danone, Renault, Siemens, Stanford University, Societe Generale, KPMG, Pasqal, Veepee, Gendarmerie Nationale, Gematek
- **Stats:** 5,249 remote conferences, 14.1M discussions, 2,478 maps built, 4,600 simultaneous users
- **Use cases:** Virtual office, virtual events, digital classroom, HR & recruitment, hybrid work
- **Integrations:** Microsoft Teams, Discord, Element, Google Workspace, Jitsi, BBB
- **Community:** Discord (7.8k+ members), X (@Workadventure_), LinkedIn
- **Updates:** Regular releases (v1.30 PWA installer, v1.31 polished UX, v1.32 noise suppression)
- **SaaS:** Hosted version at workadventu.re/pricing

### WorkAdventure Features Universe Inherits
- Proximity-based audio/video chat
- Tiled map editor with WAM format
- Scripting API (UI, player, state, room)
- Avatar system (Woka/pipoya)
- Megaphone broadcast system
- Area/zone system (entry/exit, meeting rooms, silent zones, broadcast)
- Matrix chat integration
- OIDC authentication
- Self-hosting via Docker Compose or Helm
- White label / rebranding

### What Universe Added
- AI agent system with configurable behaviors (idle, patrol, social)
- MCP server integration (streamable HTTP, OAuth, bearer, API key)
- Streaming token-by-token bot conversations
- Encrypted credential storage (AES-256-GCM)
- Bot avatar system
- Recursive self-architecture (bots spawning bots)
- Admin API for bot/MCP management (separate repo)
- PostHog observability for AI traces
- MCP OAuth with PKCE and auto-discovery

## Gather (Market Context)

- **Website:** gather.town
- **What it is:** Virtual office for remote teams — "The Delightful Virtual Office"
- **Positioning:** Built for spontaneous collaboration and "the moments in between" meetings
- **Key features:**
  - See who's free/focused/in meetings at a glance
  - Walk up and talk — no links required
  - Distraction control (Simplify View, Mini Mode, muted by default)
  - AI meeting notes included
- **Pricing:** Free 30-day trial, then paid plans
- **Scale:** 5,000+ remote teams
- **Customers:** Horaizon, hellobot, Pegasus, mutherboard, Tundr, awork
- **Competitive note:** WorkAdventure positions itself as a Gather alternative ("Migrating from gather.town? Get a discount!")
- **Relevance:** Similar market — virtual spatial environments for collaboration
