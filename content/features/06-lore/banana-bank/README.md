# Banana Bank

A Universal Basic Income vision funded by fully automated operations. Not yet implemented.

## The Vision

Profits from the zero-cost, zero-staff company model (see [power-of-zero](./power-of-zero/)) seed the world's first Banana Bank UBI program. "UBI not as theory, but as living infrastructure."
