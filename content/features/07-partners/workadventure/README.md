# WorkAdventure

Universe is a fork of WorkAdventure by TheCodingMachine — an open-source (MIT) virtual world platform. We maintain an active relationship with the upstream project.

## Credit

We credit WorkAdventure:
- On bawes.net — noted as built on WorkAdventure
- In blog posts — "Universe is an open virtual world platform built on WorkAdventure"
- This folder documents the inheritance chain
