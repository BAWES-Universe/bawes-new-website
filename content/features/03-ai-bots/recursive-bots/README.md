# Recursive Bots

> Bots spawning bots — autonomous AI agent provisioning.

**Status:** 🟢 LIVE

A manager bot can create a worker bot via the admin API. The spawned bot gets its own MCP servers, behavior, personality, and room assignment. Fully autonomous from moment of creation. When a bot can't solve a problem, it can spawn a sub-agent. A closed loop: AIs provisioning AIs on real infrastructure.
