# Technology Stack

> The tools and technologies behind Universe.

**Status:** 🟢 LIVE

### Frontend: Next.js 14, Phaser.js, Tailwind CSS, Framer Motion, Svelte
### Backend: TypeScript/Node.js, Next.js API routes, WebSocket (Pusher)
### Database: PostgreSQL (Prisma), Redis
### AI: OpenAI-compatible, LM Studio, MCP protocol
### Infrastructure: Coolify, Docker Compose, Helm, Traefik
### Observability: PostHog, Sentry
