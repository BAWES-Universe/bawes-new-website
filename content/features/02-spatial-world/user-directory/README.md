# User Directory

**Status:** 🟢 LIVE (inherited from WorkAdventure)

Browse all users in your space, see their status, and locate them on the map.

## How It Works

- Open the user list from the action bar
- See all connected users, their status (Online, Busy, etc.)
- Click on a user to locate them on the map
- Walk toward them or send a message
