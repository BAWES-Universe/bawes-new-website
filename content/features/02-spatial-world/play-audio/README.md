# Play Audio

**Status:** 🟢 LIVE (inherited from WorkAdventure)

Play audio files when players enter designated area zones. Used for ambient sound, announcements, or interactive experiences.

## How It Works

- Add a "Play Audio" property to any area in the map editor
- Upload an audio file
- When a player walks into the area, the audio plays
- Audio stops when the player leaves the area
