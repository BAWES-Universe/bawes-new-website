# Lock Discussion

**Status:** 🟢 LIVE (inherited from WorkAdventure)

Lock your conversation bubble to prevent others from joining. Useful for private conversations in public spaces.

## How It Works

- In an active conversation bubble, click the lock button
- No new participants can enter the bubble
- Existing participants remain
- Unlock to allow others to join again
