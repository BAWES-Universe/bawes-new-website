# Text Chat

In-world text messaging. Every room has its own chat channel. Universe extends this with bot streaming.

## Features

- Per-room message history
- Chat bubble display in game UI
- Proximity-based visibility (nearby players see your messages)

## Bot Streaming (Universe Extension)

AI bot responses stream token-by-token into the chat bubble in real time. See [bot-streaming-chat](../03-ai-bots/bot-streaming-chat/) for details.
