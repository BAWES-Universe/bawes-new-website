# Megaphone

Broadcast your audio, video, or screen to every player in a room or an entire world. Typically reserved for admins and moderators.

## How It Works

- Click megaphone icon in the action bar to enable
- Orange icon + yellow camera feedback + megaphone icon on avatar = broadcasting
- Scope: "room" (current room) or "world" (all rooms sharing the same space name)
- Access controlled by tags — only tagged users see the megaphone button

## Configuration

Configured in [inline map editor](./inline-map-editor/) under the megaphone tab. Toggle on/off, set scope, set access tags.
