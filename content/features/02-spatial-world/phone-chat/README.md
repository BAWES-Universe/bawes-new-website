# Phone Chat

**Status:** 🟡 PLANNED

Direct audio/video call between any two users, working across different rooms — not proximity-dependent.

## Planned Implementation

- Call any other user from the interface
- Full audio/video, like a traditional video call
- No proximity requirement (unlike [proximity-chat](./proximity-chat/))

> **Note:** Phone Chat is documented here as a planned feature. It is not yet implemented in the current codebase. For real-time communication, use [Proximity Chat](./proximity-chat/) or [Text Chat](./text-chat/).
