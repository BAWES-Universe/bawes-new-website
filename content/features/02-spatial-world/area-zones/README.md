# Area Zones

> Spatial zones on maps that trigger actions when players enter, exit, or stay.

**Status:** 🟢 LIVE — inherited from WorkAdventure

### Zone Types
| Zone | What it does |
|---|---|
| Entry/Exit | Triggers on player enter/leave |
| Meeting Rooms | Jitsi/BBB zone for scheduled calls |
| Silent Zones | Mute audio/video in designated areas |
| Broadcast Zones | Push notifications to players in area |
| Restricted Areas | Tag-based access control |
