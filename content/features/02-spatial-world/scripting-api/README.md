# Scripting API

> Programmatic control of the virtual world via the WA.* namespace.

**Status:** 🟢 LIVE — inherited from WorkAdventure

### API Surface
| API | What it controls |
|---|---|
| WA.room | Room state, layers, events |
| WA.state | Persistent key-value store |
| WA.player | Player position, tags, UUID |
| WA.ui | Popups, menus, iframes, modals, action bar |
| WA.chat | Send/receive chat messages |
| WA.mapEditor | Programmatic map editing |

Supports popups, custom menus, iframes, modals, action bar buttons, and remote player interaction.
