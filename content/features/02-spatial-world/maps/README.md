# Maps

> Map creation and publishing system using Tiled editor + WAM format.

**Status:** 🟢 LIVE — inherited from WorkAdventure

### How It Works
- Editor: Tiled (desktop map editor)
- Format: WAM (Web Avatar Map — JSON schema)
- Layers: Object layers for scripting triggers, collision detection
- Publishing: WA hosted, GitHub Pages, or self-hosted

### Map Features
- Special zones (entry/exit, meeting rooms, cameras, animations)
- Embedded websites as map objects
- Text displays on map
- Custom object properties
