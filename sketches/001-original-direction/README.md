## Design Direction: Living Space

### What's different
Not Stripe. Not a template. This direction treats Universe as what it really is — a **living spatial world** where people and AI coexist.

### Key choices
- **Color**: Deep indigo background, purple→amber→purple shimmer gradient as the hero accent. Warm amber/gold (#f59e0b) is a new accent that ties back to the BAWES logo.
- **Hero**: Animated scene showing floating avatars with a bot speaking to the user, connected by SVG path animations. Shows the product instead of describing it.
- **Bento layout**: Varied card sizes (2-col wide + 1-col narrow) creating visual rhythm. Cards have hover-revealed CTAs.
- **Typography**: Space Grotesk for display headings (bold, tight tracking), Inter for body text. Clean hierarchy.
- **Cards**: Subtle glass at `rgba(255,255,255,0.03)` with `backdrop-filter: blur(12px)`. Purple hex grid background pattern in the hero scene.
- **Tone**: Direct, almost challenging. "Not a chat app. Not a game." Acknowledges what people assume before correcting it.

### What changed from V4
- Background: #0a0a14 → #0a0814 (slightly warmer, deeper indigo)
- Primary purple: #8b5cf6 → #a78bfa (lighter, more luminous)
- New accent: #f59e0b amber/gold (warmth, ties to logo)
- Headline font: Inter → Space Grotesk (more distinctive)
- Glass opacity: 0.04 → 0.03 (more subtle)
- Border: `rgba(139,92,246,0.15)` → `rgba(255,255,255,0.06)` (more neutral)
- Landing page: uniform card grid → bento layout
- Hero: text-only → animated scene with avatars
- Added noise overlay for texture
